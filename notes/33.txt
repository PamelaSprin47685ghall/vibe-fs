Methodology 方法论系统：
MethodologySchema = { methodologyId; toolName; shortDefinition; triggerWhen; toolDescription; fields; meditatorRole; outputSections }
MethodologyField = { name; description; required; kind: String|StringArray; minArrayItems }
buildSchema: methodologyId + fields → MethodologySchema，自动生成 toolName = "methodology_" + id。
toToolCatalogSpec: MethodologySchema → ToolSpec，注册到 ToolCatalog。
每个方法论对应一个 notebook 工具，共 54 个。
allSchemas: 54 个方法论模式列表，在 Methodology.Registry 中注册。
SchemaCommon.renderInputYaml: 字段值 → YAML，供 meditator 读取。
renderMeditatorIntent: schema + YAML → 完整方法论提示。
每个工具调用创建一个 meditator 子会话执行方法论推理。
