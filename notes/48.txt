Shell WorkspaceFiles 能力文件发现：
findCapsFiles: 读取 AGENTS.md → 解析 YAML front-matter import 列表 → 递归收集文件。
Budget: 2000 文件上限，8MB 总大小上限，4MB 单文件上限。
ReverieFileResult: { filePath; content: option; skipReason: option }
readReverieFiles: 读取多个文件供 meditator 使用。
splitFrontMatter: 分离 YAML front-matter 与 body。
extractImportList: 从 front-matter 提取 import 列表。
OmpCaps 变体 (Shell.OmpCaps):
正则匹配大写文件名 (^[A-Z][A-Z0-9_]*\.md$) + 目录递归。
排除 AGENTS/CLAUDE/NODE_MODULES 等目录。
XML 格式输出 <caps-context file="..." />。
