Executor 核心：
ExecutorLanguage = Shell | Python | Javascript
ExecutorTimeoutType = Short(1s) | Long(10s) | LastResort(100s)
ExecuteOptions = { program; language; dependencies; timeoutType; mode; cwd; whatToSummarize }
ExecuteResult = Completed | Truncated | Failed | MissingExecutable
summaryThresholdBytes = 8192，超限自动摘要。
readOnlyReadCommands: head/tail/sed/cat/grep/rg/find 等，检测到则追加安全警告。
shouldAppendReadOnlyWarning: 检查命令中是否含只读命令。
prepareShellProgram: 剥离管道后执行。
buildSummaryPrompt: 超限时调用子代理摘要。
