FuzzyPath 路径解析：
normalizePathConstraint: path + cwd → 相对路径约束（安全规范化）。
resolveFuzzySearchPath: inputPath + cwd → { basePath; pathConstraint; external }。
external = true 时搜索外部路径（basePath 非 cwd）。
normalizeExcludes: exclude 模式规范化，支持 ! 前缀排除。
buildQuery: path constraint + pattern + exclude → 搜索查询字符串。
resolveExternalBasePath: 绝对路径 → (base, constraint)，处理文件/目录。
isOutside: 检查目标是否在 basePath 之外，防止路径穿越。
FuzzyQuery:
detectGrepMode: pattern → "plain" | "regex"。
checkWildcardOnly: 纯通配符模式拒绝（防止全量匹配）。
FuzzyFindParams / FuzzyGrepParams: 查询参数类型。
