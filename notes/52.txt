ToolExecutionContext = { Directory; SessionId; WorkspaceId }
fromMuxConfig: config → IToolRuntimeContext，workspaceId 必填。
fromOpencode: context + fallbackDir → IToolRuntimeContext。
pluginDirectoryFromCtx: 从 opencode ctx 提取目录。
sessionId/workspaceId/sessionIdString/workspaceIdString: 查询函数。
ToolContextCodec:
decodeOpencodeToolContext: context 对象 → ToolExecutionContext，多 key 容错查找。
decodeMuxConfig/decodeMuxConfigLenient: 严格/宽松两种解析模式。
muxConfigDirectoryFallback: Mux config 目录回退。
