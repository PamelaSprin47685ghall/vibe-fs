BacklogProjection 回溯投影：
BacklogEntry = { report }
isTodoResultFor: ToolPart 中 toolName = todowrite/task 且 status = completed。
isTodoErrorFor: status = error。
findFoldRange: 在 flat parts 中定位折叠范围 — 至少 N 个 todo 锚点。
buildSyntheticPrefixMessages: 折叠回溯 → 合成 user 前缀消息。
rebuildVisibleOnly: 仅保留可见 flat part 对应的消息部分。
projectBacklogFor: 核心折叠函数 — 找到折叠范围，折叠中间 todo，合成前缀，重建消息列表。
foldAfterFirst: 控制折叠策略 — true 保留第一个 todo 结果。
BacklogProjectionCore:
buildBacklogText: 回溯条目 + 用户提示 → YAML front-matter 格式的回溯文本。
renderCompletedWorkEntry: 用户消息列表 + 报告 → front-matter 项。
