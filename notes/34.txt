Methodology.Args 参数解析：
parse: schema + obj → Result<(scalar Map, array Map), string>
FieldKind.String: 空字符串检查 required，非 required 空值跳过。
FieldKind.StringArray: 检查 isArray，required 检查 minArrayItems。
验证在 Kernel 层纯函数完成，与宿主无关。
Methodology 工具在三个宿主中注册方式：
MuxTools: JSON Schema 参数 + execute → runMuxSubagent。
OmpTools: typebox 参数 + execute → runSubagent。
OpencodeTools: Zod 参数 + execute → runSubagentCoreResult。
三宿主共享 SchemaCommon 定义的方法论模式。
