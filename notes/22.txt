KnowledgeGraph Types:
KnowledgeGraphId = KnowledgeGraphId of string (4 位 hex)
KnowledgeGraphEntry = { id; entity: string list; fact }
KnowledgeGraphDraft = { id: string option; entity: string list; fact }
KnowledgeGraphHeader = DayHeader(date, rewritten)
KnowledgeGraphFile = { header; entries }
KnowledgeGraphProjection = Map<KnowledgeGraphId, KnowledgeGraphEntry>
KnowledgeGraphJobKind = AppendAfterWork | DailyRewrite(date)
KnowledgeGraphJobContext = { workspaceRoot; kind }
核心原则：事件溯源 — NDJSON 文件是事实源，projection 是从文件折叠出的当前态。
